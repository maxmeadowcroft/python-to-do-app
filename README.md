# Python To-Do List App

This is a simple command-line To-Do List application.

## Features
- Add tasks
- View tasks
- Mark tasks as complete
- Delete tasks

## How to Run
1. Clone the repository:

```Shell
git clone https://github.com/maxmeadowcroft/python-to-do-app.git
```

2. Run the app:

```Shell
python todo.py 
```
